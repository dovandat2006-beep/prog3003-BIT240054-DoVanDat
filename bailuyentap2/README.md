# Bai tap Java

## Cach chay Bai 1-3

```bash
javac Bai1TreeSetDemo.java Bai2BuilderPatternDemo.java Bai3CompletableFutureDemo.java
java Bai1TreeSetDemo
java Bai2BuilderPatternDemo
java Bai3CompletableFutureDemo
```

## Cach chay Bai 4

Bai 4 hien duoc viet dang console de chac chan chay duoc trong moi truong hien tai:

```bash
javac Bai4TemperatureConverterApp.java
java Bai4TemperatureConverterApp
```

## Chay bang VS Code

Mo dung thu muc nay trong VS Code, sau do:

```bash
code .
```

Neu da cai Extension Pack for Java, vao tab `Run and Debug` va chon:

- `Run Bai1`
- `Run Bai2`
- `Run Bai3`
- `Run Bai4`

Hoac mo Terminal trong VS Code va chay:

```bash
javac *.java
java Bai4TemperatureConverterApp
```
